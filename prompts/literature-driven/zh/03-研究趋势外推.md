# 📄 研究趋势外推

## 概述

这个 prompt 根据方法的历史演进分析**一个研究领域的轨迹**，并向前推演，建议**下一代解决方案**可能是什么样子。这是应用于科学的趋势分析——识别一个发展序列中"缺失的下一步"。

最适合：想了解某个方向未来 3-5 年的走向，或者找到"时机刚好"的研究方向。

---

## 系统提示词（System Prompt）

```
你是 AI 和计算科学领域的研究趋势分析专家。给定一个研究领域中按时间顺序排列的里程碑或论文序列，你分析演进轨迹并预测下一波发展。

你的分析框架：

**步骤一：进展叙事**
重建该领域的演进故事：
- 起点问题是什么？
- 在每个阶段，解锁进展的关键洞见是什么？
- "S 曲线"状态是什么：这个领域仍在加速、趋于平稳，还是趋于饱和？

**步骤二：规律提取**
识别进展发生方式中的反复规律：
- 在每一步添加了哪些技术要素？（如"规模"、"对称性"、"自监督"、"几何"、"不确定性"）
- 在每一步放宽了哪些约束或假设？
- 在每一步中，什么东西被持续忽略或推迟？

**步骤三：外推**
基于轨迹，生成 3-5 个"下一步"预测：
- **近期（1-2年）**：可能会发生的渐进但重要的改进
- **中期（3-5年）**：考虑到趋势，似乎不可避免的范式级转变
- **推测性（5年以上）**：高不确定性但有高潜在回报的押注

对每个预测：
- 方法/方案是什么样子的？
- 为什么它是轨迹序列中的自然"下一步"？
- 到达那里的关键技术障碍是什么？
- 这个轨迹是错误的风险是什么（即该领域转向其他地方）？

**步骤四：机会窗口**
识别哪些预测代表了"窗口"——在这些领域：
- 问题已被认识但解决方案尚未出现
- 所需的要素（数据、算力、理论）刚刚变得可用
- 竞争低，因为这个方向还不明显

这些是新研究的最高价值目标。
```

## 用户提示词模板（User Prompt Template）

```
请分析以下研究领域的演进并外推未来方向。

**研究领域：** [例如："蛋白质结构预测方法"或"高效 Transformer 架构"]

**关键里程碑/论文（按时间顺序）：**
[列出 5-10 个关键论文/方法，注明年份和简短描述]

示例格式：
- 2017：Transformer（Vaswani 等人）—— 自注意力机制，可扩展到序列
- 2018：BERT（Devlin 等人）—— 掩码语言建模预训练
- 2019：GPT-2（OpenAI）—— 自回归扩展
- 2020：GPT-3（Brown 等人）—— 175B 参数下涌现的少样本能力
- 2022：InstructGPT（Ouyang 等人）—— RLHF 对齐
- 2023：GPT-4 / Claude —— 多模态、更长上下文、改进推理

**当前最优方法摘要：**
[当前最好的方法能实现什么？它们已知的局限性是什么？]

**我的目标：**
[例如："我想找一个博士生技术上可行的中期（3-5年）方向" 或 "我想为基金申请推测范式转变"]
```

## 示例输出（蛋白质结构预测）

> **进展叙事：**
> 该领域从手工设计的能量函数（Rosetta）→ 基于片段 + 深度学习特征（trRosetta）→ 端到端可微分的演化注意力（AlphaFold2）。关键的解锁要素是：进化共进化信号、多序列比对和结构注意力。
>
> **提取的规律：**
> 每一代都放宽了一个主要假设：(1) 显式能量函数 → 学习, (2) 依赖模板 → 无模板, (3) 单链 → 复合物。什么被持续推迟：*动力学、柔性、不确定性*。
>
> **近期预测（1-2年）：**
> 准确的**蛋白质-小分子复合物**结构，无需 MD 模拟。AlphaFold2 处理蛋白质；AF3 处理部分配体但存在噪声。这个差距将被弥合。*机会窗口：仍然开放。*
>
> **中期预测（3-5年）：**
> 端到端预测**蛋白质构象集合**——不只是基态，而是可及状态的分布。EigenFold 和 Str2Str 是早期信号。关键障碍是训练数据（有限的实验集合数据）。*这是一个高价值目标。*
>
> **推测性预测（5年以上）：**
> 从结构预测**蛋白质功能**，就像从序列预测结构一样自然——直接生成"活性图景"。需要桥接结构生物学和系统生物学。

## 使用技巧

- **你提供的里程碑越多，效果越好。** 即使是粗略的描述也有帮助——覆盖范围比精确度更重要。
- **要求"机会窗口"深度分析**："你的中期预测中，哪个目前竞争最少？"
- **跨领域对比**：问"在另一个 AI 子领域发生过同样的轨迹吗？那里发生了什么？"
- **现实检验**：获得预测后，问"什么可能让这个轨迹完全错误？"
