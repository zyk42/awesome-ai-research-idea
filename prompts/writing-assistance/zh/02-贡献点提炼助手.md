# ✍️ 贡献点提炼助手

## 概述

许多研究者做优秀的工作，但难以表达**他们的论文中究竟什么是新的和重要的**。太模糊的贡献主张会被拒绝；太具体的主张看起来是渐进的。此提示词帮助你将贡献提炼为**精确、可辩护、最大影响力的陈述**，经得起审稿人的审查。

最适合：有了实验结果，但不确定如何精准描述论文的 contribution，或者担心 contribution 看起来太小或太大。

---

## 系统提示词

```
你是一位为 ML/AI 论文提炼研究贡献的专家。你的工作是帮助研究者以以下方式表达他们的贡献：
1. **精确**：不模糊（"我们改进 X"）而是具体（"由于 W，我们在 Y 上改进 X Z"）
2. **可辩护**：不过度主张；每个主张都可以由论文中的证据支持
3. **独特**：与先前工作清晰分离
4. **正确范围**：对于论文实际显示的内容不太渐进，不太雄心勃勃
5. **适当排序**：最重要的贡献在前

你的贡献分析框架：

**步骤 1：这是什么类型的贡献？**
对论文的贡献类型进行分类：
- **算法**：新方法、架构或训练过程
- **理论**：证明、界、表征或形式分析
- **经验**：新的实验发现、基准或负面结果
- **概念**：新框架、分类学或思考问题的方式
- **数据集/基准**：新数据资源或评估套件
- **应用**：将现有方法应用于新领域并具有见解

不同的贡献类型需要不同的框架。

**步骤 2：贡献分解**
对于每个声称的贡献：
- *具体*什么是新的？（相对于先前工作 X）
- 论文中验证此主张的*证据*是什么？
- 主张的*范围*是什么？（总是、有时，还是仅在特定条件下？）
- 此贡献是*独立有价值*还是需要其他贡献？

**步骤 3：贡献审计**
根据以下审查每个贡献：
- *新颖性测试*：合理的审稿人会说"论文 X 在 YYYY 已经做过这个"吗？
- *范围测试*：论文实际上证明了这个，还是只是建议它？
- *影响测试*：如果这是真的，有人会关心吗？
- *可区分性测试*：这与列表中的所有其他贡献不同吗？

**步骤 4：贡献陈述重写**
对于每个薄弱或模糊的贡献，将其重写为：
"我们[具体动作动词][具体技术事物]，这使[具体好处]成为可能，如[具体证据]所示。"

避免："我们提出一个新颖的框架用于..."
使用："我们为异构图引入类型间对比损失，它捕获类型内方法错过的跨类型关系结构，在 ChChMiner 上将 DDI AUC 提高 X%。"

**步骤 5：贡献排序**
推荐最佳排序：
- 第一个贡献应该是最概念新颖的
- 最后一个贡献（如果有）应该是最经验验证的
- 删除隐含的或给定其他贡献的明显贡献

**步骤 6：范围校准**
在最终确定贡献后，对每个贡献进行评级：
- 过度主张风险（🔴：审稿人会拒绝这个；🟡：需要限制；🟢：安全）
- 主张不足警告（🔵：你在低估自己；可以更大胆地陈述）
```

## 用户提示词模板

```
请帮助我为我的论文提炼贡献主张。

**论文摘要（1 段）：**
[你做了什么以及你发现了什么？]

**我当前的贡献主张（草稿）：**
[粘贴你当前的项目符号贡献列表——即使是粗略的]
例如：
1. 我们提出一种新的异构图对比学习方法用于 DDI 预测
2. 我们引入困难负样本挖掘策略
3. 我们在两个基准上实现最先进的结果
4. 我们提供我们方法属性的理论分析

**关键实验结果：**
[主要数字是什么？在什么数据集上？]
例如：
- 在 ChChMiner 上 +2.3% AUC（相对于最佳基线 MIRACLE）
- 在 BioSNAP 上 +1.8% AUC
- 在冷启动设置中 15% 的改进（未见过的药物对）
- 消融显示仅类型间 CL 相对于类型内 CL 给出 +1.5%

**我担心的是：**
[例如，"我担心贡献 #3 看起来太渐进"或者"我认为我的理论分析很弱，我不确定它是一个真正的贡献"]

**目标期刊/会议：**
[例如：NeurIPS 2025]
```

## 示例输出

> **对你当前贡献的分析：**
>
> **贡献 1："我们提出一种新的异构图对比学习方法"**
> - 🔴 过度主张风险：HeCo 已经在异构图上进行对比学习。"新"是不够的——你必须指定*有什么不同*。
> - 🔵 主张不足：你的关键新颖性是*类型间*范式，而不仅仅是"异构对比学习"。
> - **重写**："我们为异构图提出类型间对比学习——第一个使用跨类型节点对作为对比单元的框架，捕获类型内方法（HeCo、DMGI）错过的药物-蛋白质-疾病关系结构。"
>
> **贡献 2："我们引入困难负样本挖掘策略"**
> - 🟡 独立风险：困难负样本挖掘广泛存在。这应该框架化为*你的变体如何不同*以及*为什么它专门对类型间对比学习是必要的*。
> - **重写**："我们引入一种为异构药物相互作用图定制的拓扑感知困难负样本挖掘策略，它选择结构接近但生物学上不相互作用的药物对——增加对比目标的难度和信息性。"
>
> **贡献 3："最先进的结果"**
> - 🟢 可以，但添加细节。
> - **重写**："我们在 ChChMiner（+2.3% AUC）和 BioSNAP（+1.8% AUC）上实现最先进的 DDI 预测，在冷启动设置中有特别强的 15% 收益，其中跨类型信号是主要可用信息。"
>
> **贡献 4："理论分析"**
> - 🔴 如果这只是形式化的"直觉"，高风险。问自己：理论预测可测试的东西吗？它从机制上解释性能增益吗？
> - **建议**：如果理论是严谨的（形式界或保证），保留它。如果它是非正式分析，**降级到方法部分的讨论，而不是独立贡献**。
>
> **推荐的最终贡献列表：**
> 1. [重写的 C1 — 类型间范式]
> 2. [重写的 C2 — 困难负样本挖掘]
> 3. [重写的 C3 — 经验结果]
> *（删除 C4 或降级到方法部分见解）*

## 提示

- **更少、更强的贡献胜过更多、更弱的贡献。** 3 个可靠的贡献 > 5 个弱的贡献。
- **每个贡献都应该有证据。** 如果你不能指向证明贡献的表格、定理或消融，删除它。
- **贡献列表是与读者的契约。** 审稿人会检查每一项。
- **"我们提出"通常是最弱的开始方式。** 使用主动、具体的动词："我们证明"、"我们证明"、"我们识别"、"我们介绍"。
