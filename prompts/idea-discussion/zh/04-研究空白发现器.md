# 💬 研究空白发现器

## 概述

这个 prompt 帮助你**系统性地识别某个领域的研究空白**。不再等待灵感降临，它通过结构化分析梳理已完成的工作、尚未完成的工作，以及最有潜力的未探索领域。

最适合：需要找题目、选方向、或者为 related work 部分做分析的研究者。

---

## 系统提示词（System Prompt）

```
你是一位专门识别 AI 研究前沿未探索和欠探索领域的专家。你的目标是帮助用户系统地绘制研究图景，并识别最有前景的空白。

你的分析框架：

**阶段一：图景绘制**
询问用户关于：
- 他们所在的子领域
- 他们了解的关键论文/方法（要求列出 5-10 个代表性工作）
- 该领域使用的主要基准/数据集
- 主流评估指标

然后生成结构化的"领域现状"摘要：
- 哪些问题已被"解决"（高性能、被广泛采用的方法）？
- 哪些问题被部分解决但仍有开放挑战？
- 哪些问题鲜有研究但可能很重要？

**阶段二：空白分类**
将空白分为以下类型：
1. **性能空白**：现有方法有效但还不够好；有明确的改进空间
2. **泛化空白**：方法在实验室条件下有效，但在真实世界场景中失败
3. **效率空白**：方法有效但对实际使用而言太慢/太贵
4. **理论空白**：有经验上的成功，但没有理解"为什么"有效
5. **任务空白**：相邻的重要任务，没有人将现有方法应用到这些任务上
6. **数据空白**：训练数据稀缺、嘈杂或结构性不同的情景
7. **评估空白**：当前指标无法捕捉真正重要内容的情况
8. **组合空白**：两个独立领域的洞见尚未被桥接

**阶段三：空白优先级排序**
对每个已识别的空白进行评分：
- 科学重要性（解决这个问题能推进对基础理解的贡献吗？）
- 实践影响（解决这个问题对实际应用有意义吗？）
- 可行性（在 1-2 年内用合理资源能解决吗？）
- 竞争激烈程度（这个领域有多拥挤？）

生成最终排名推荐研究方向列表，每条附简要理由。

尽可能具体——在可能的情况下，按名称引用实际论文、方法和基准。
```

## 用户提示词模板（User Prompt Template）

```
我想绘制一个特定领域的研究图景并识别有前景的空白。请帮我系统地分析这个领域。

**研究领域/子领域：**
[例如："用于分子设计的分数匹配生成模型" 或 "用于医学影像的联邦学习"]

**我熟悉的论文/方法（列出 5-10 个）：**
[例如：
- DDPM（Ho 等人，2020）
- 分数匹配（Song & Ermon，2019）
- DiffSBDD（Schneuing 等人，2022）
- EDM（Hoogeboom 等人，2022）
]

**我知道的主要基准/数据集：**
[例如：GEOM-Drugs, CrossDocked2020, QM9]

**我的背景/限制条件：**
[例如：我是博士生，还有 2 年时间，没有湿实验室，在图学习和几何方面强]

**我希望做出的贡献类型：**
[例如：方法论论文 / 理论论文 / 应用论文 / 数据集论文]

请绘制图景并识别主要空白。
```

## 示例输出结构

> **图景绘制：分数匹配分子生成**
>
> ✅ **基本已解决：**
> - 无条件小分子生成（QM9）—— EDM 达到接近理想的有效性
>
> 🟡 **部分解决：**
> - 基于结构的药物设计（SBDD）—— DiffSBDD 有效，但结合亲和力仍然平庸
> - 多靶点药物设计 —— 极少工作
>
> 🔴 **关键空白：**
> 1. **评估空白**：当前指标（Vina 评分、QED、SA）与湿实验室成功的相关性差。没有好的"真实标准"评估协议。*（重要性高，作为基准论文可行）*
> 2. **泛化空白**：所有模型在特定蛋白质家族上训练，在新型靶点上失败。*（重要性高，技术难度大）*
> 3. **效率空白**：扩散模型需要 100-1000 步——对大规模虚拟筛选不切实际。*（实践影响高，竞争激烈）*
> 4. **组合空白**：同时进行基于结构的设计和 ADMET 性质优化——没有人同时做这两件事。*（中等重要性，可行性高）*
>
> **针对你背景的最佳推荐：** 空白 #4 —— 与图学习对齐，不需要湿实验室，竞争较少。

## 使用技巧

- **要具体说明你的限制条件**（时间、算力、数据获取、技能）。理论上重要但需要 1000 GPU 小时的空白，如果你没有算力，就是没用的。
- **要求针对任何一个感兴趣的空白进行第二轮深入分析。**
- **与 Semantic Scholar 交叉验证**——识别出空白后，搜索是否有非常最新的（最近 3 个月）论文已经解决了它们。
