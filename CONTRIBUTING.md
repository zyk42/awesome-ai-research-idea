# Contributing to Awesome AI Research Idea

Thank you for your interest in contributing! This project grows stronger with every new prompt.

## How to Add a New Prompt

1. **Choose the right category** (or propose a new one)
2. **Follow the prompt template format** — each file must include:
   - `# Title`
   - `## Overview` — what this prompt does in 2-3 sentences
   - `## System Prompt` — the full system prompt in a code block
   - `## User Prompt Template` — the starting message template
   - `## Example Output` (optional but encouraged)
   - `## Tips` — how to use it effectively
   - `## Tags` — relevant keywords

3. **Name your file** using the pattern: `XX-short-kebab-case-name.md`

4. **Submit a PR** with:
   - The new `.md` prompt file
   - An update to the relevant `README.md` table

## Prompt Quality Standards

- ✅ Prompts should be **specific and actionable**
- ✅ System prompts should define a **clear AI persona and role**
- ✅ User templates should have **obvious fill-in points** marked with `[brackets]`
- ✅ Prompts should be **tested** — include an example or note the model you tested with
- ❌ Avoid vague instructions like "help me with research"
- ❌ Avoid prompts that only work with one specific AI tool

## Categories

| Folder | Description |
|---|---|
| `idea-discussion/` | Open-ended idea development and Socratic dialogue |
| `literature-driven/` | Input: papers/abstracts → Output: new ideas |
| `critical-review/` | Adversarial review and assumption challenging |
| `experiment-design/` | Planning experiments, metrics, and ablations |
| `interdisciplinary/` | Cross-domain AI + science fusion |
| `writing-assistance/` | Paper structure, contributions, framing |

## Code of Conduct

Be respectful, be scientific, be curious. This is a community for researchers helping researchers.
