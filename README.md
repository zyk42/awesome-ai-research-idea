## рџљЂ How to Use

1. Browse the category that matches your current need.
2. Open the prompt `.md` file.
3. Each file contains:
   - **[SYSTEM PROMPT]** вЂ” paste this into the system/instructions field of your AI tool
   - **[USER PROMPT TEMPLATE]** вЂ” fill in the `[brackets]` and send as your first message
   - **Tips** on how to get the most out of the conversation
4. Iterate! The best results come from multi-turn conversations.

> **Recommended AI tools:** ChatGPT (GPT-4o), Claude, Gemini Advanced, DeepSeek-R1, Kimi, or any capable LLM with system prompt support.
