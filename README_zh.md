## ✨ 这是什么？

**Awesome AI Research Idea** 是一个 prompt 工程工具箱，专为希望将 AI 作为真正智识合作者的研究者设计——而不仅仅是一个搜索引擎。

无论你是在头脑风暴新算法、压力测试一个假设，还是从一堆论文中寻找灵感，本仓库的 prompt 都能帮助你与 AI 进行**更深入、更结构化、更高效**的对话。

## 🚀 如何使用

1. 浏览符合你当前需求的分类。
2. 打开对应的 prompt `.md` 文件。
3. 每个文件包含：
   - **[SYSTEM PROMPT]** — 粘贴到 AI 工具的系统提示/指令栏
   - **[USER PROMPT 模板]** — 填写 `[方括号]` 中的内容，作为第一条消息发送
   - **使用技巧** — 帮助你最大化对话效果
4. 多轮迭代！最好的结果来自多轮深度对话。

> **推荐 AI 工具：** ChatGPT (GPT-4o)、Claude、Gemini Advanced、DeepSeek-R1、Kimi，或任何支持系统提示的强大 LLM。
